Metrika Goal Tracker

Metrika Goal Tracker — это расширение Chrome для проверки установки Яндекс.Метрики и отслеживания JavaScript-целей (reachGoal) прямо в DevTools браузера.

Расширение предназначено в первую очередь для маркетологов, аналитиков и специалистов по контекстной рекламе, которым важно быстро понять:

установлен ли счётчик Метрики на сайте;
какой именно счётчик работает;
какие JavaScript-цели реально срабатывают при взаимодействии с сайтом.
